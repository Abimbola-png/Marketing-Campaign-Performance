# Marketing-Campaign-Performance
This project showcases a Power BI dashboard built to analyze 1,000 marketing campaigns across different products and channels.

The data was cleaned and structured to ensure accuracy, with key metrics such as ad spend, impressions, clicks, conversions, and revenue analyzed.

Key performance indicators include:

CTR: 5.09%
Conversion Rate: 9.81%
ROI: 141.51

The dashboard features visuals like channel performance, revenue by product, and time trends, along with interactive filters for deeper analysis.
